## Page Breakdown

### 1. index.html - Home/About Me
- **Hero Section**: Animated background with professional introduction
- **About Me**: Personal story, career goals, and key strengths
- **Quick Stats**: Education, skills count, projects completed
- **Call-to-Action**: Links to other sections

### 2. cv.html - Professional CV
- **Career Objective**: Professional summary and goals
- **Technical Skills**: Programming languages, frameworks, tools
- **Soft Skills**: Problem-solving, teamwork, communication
- **Education**: AIUB degree details and coursework
- **Experience**: Tutoring and leadership roles
- **Extracurricular Activities**: Event organization and club membership

### 3. skills.html - Skills Showcase
- **Technical Skills Grid**: Interactive skill cards with proficiency levels
- **Programming Languages**: Java, C++, Python, C#, JavaScript
- **Web Development**: HTML, CSS, React.js
- **Database Technologies**: SQL, PL/SQL, Oracle, MySQL
- **Tools & Frameworks**: Git, VS Code, Agile
- **Soft Skills**: Teamwork, communication, problem-solving

### 4. projects.html - Project Portfolio
- **Bus Ticket Management System**: Java-based booking system
- **School Management System**: Desktop application for student records
- **Project Cards**: Interactive cards with descriptions, technologies, and features
- **Code Samples**: Links to repositories (when available)

### 5. contact.html - Contact Information
- **Contact Form**: Professional inquiry form
- **Contact Details**: Phone, email, location, LinkedIn
- **Availability Status**: Open to opportunities
- **Social Links**: Professional networking profiles

### 6. certificates.html - Achievements
- **Academic Certificates**: Degree and coursework achievements
- **Online Certifications**: Professional development courses
- **Awards & Recognition**: Academic and extracurricular honors
- **Upload Areas**: Placeholders for future certificates

## Design Philosophy
- **Color Scheme**: Professional blues and grays with accent colors
- **Typography**: Clean, modern fonts for readability
- **Layout**: Responsive grid system with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, progress bars
- **Visual Hierarchy**: Clear section divisions and content organization

## Technical Features
- **Responsive Design**: Mobile-first approach
- **Smooth Animations**: CSS transitions and JavaScript effects
- **Interactive Components**: Skill bars, project filters, contact form
- **Performance Optimized**: Fast loading with optimized images
- **SEO Friendly**: Proper meta tags and semantic HTML
